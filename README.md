# Vance & Sterling | Premium Law Firm UI

A high-end, fully responsive frontend architecture designed for a top-tier law firm. Built to demonstrate advanced UI/UX principles, modern React practices, and performance optimization.

## 🚀 Tech Stack
* **Framework:** React + Vite
* **Styling:** Tailwind CSS (v4)
* **Icons:** SVG Integration
* **Architecture:** Component-based, Single Page Application (SPA)

## ✨ Key Features
* **Elite UI/UX:** Deep navy and champagne color palette to establish immediate trust and authority.
* **Dynamic Data Rendering:** Uses a mock data architecture to efficiently map out Practice Areas, Team Members, and Client Results without hardcoding.
* **Interactive State:** Custom Contact Form that utilizes React Hooks (`useState`) to simulate a secure CRM submission and trigger demo notifications.
* **Responsive Design:** Flawless scaling from mobile devices to ultrawide desktop monitors.
